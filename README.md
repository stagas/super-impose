### SuperImposeElement

**Extends HTMLElement**

Super imposes one child over another.

```js
import { SuperImposeElement } from 'super-impose'
customElements.define('super-impose', SuperImposeElement)
```

```html
<super-impose>
  <div></div> <!-- leader -->
  <div slot="follower"></div>
</super-impose>
```
